ScaleFinder: Jazz Practice Tool

Ever wondered what scale to use when improvising over a C#m7b5? Scalefinder is a desktop app that allows you to input chord symbols and view options for scales to play over each symbol, with pitches displayed on a visual piano. It was built in C++ using the graphics library Cinder.

AudioKit Piano Visualizer

After building the visual piano for ScaleFinder, I decided to make a full, 88-key visualizer in SwiftUI for a hackathon in summer 2020. I used the awesome library AudioKit to add real-time MIDI input support and audio playback. It only runs on MacOS but the source code is available below!
